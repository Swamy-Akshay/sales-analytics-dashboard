# 📊 Sales Analytics Dashboard (Power BI)

## 🚀 Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance and business insights across multiple dimensions such as time, region, and customer segments.

The dashboard enables quick decision-making by transforming raw data into meaningful visual insights.

---

## 🎯 Key Features

* 📈 Sales trend analysis over time
* 🌍 Region-wise performance breakdown
* 🧑‍🤝‍🧑 Customer segmentation insights
* 💰 Revenue and KPI tracking
* 🔍 Interactive filters and drill-down capabilities

---

## 🧠 Key Insights (Business Value)

* Identified top-performing regions contributing to majority revenue
* Detected seasonal trends in sales patterns
* Highlighted underperforming segments for strategic improvement
* Provided actionable insights for business decision-making
